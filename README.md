# AWS Static Website

My first static website hosted on Amazon S3.

## Project Overview

This project is a simple personal landing page built with HTML and CSS, then deployed using Amazon S3 Static Website Hosting.

## Live Demo

http://johny-939390172601-ap-southeast-7-an.s3-website.ap-southeast-7.amazonaws.com/

## Technologies Used

- HTML
- CSS
- Amazon S3
- AWS Static Website Hosting
- GitHub

## Challenges

- Understanding S3 bucket policy
- Debugging image path issues
- Learning HTML/CSS structure

## Features

- Static website hosting on Amazon S3
- Public bucket policy for website access
- Custom CSS styling
- Image asset management
- Button hover and active effects
- Fade-in animation
